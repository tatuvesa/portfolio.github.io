My Projects

Project 1

This project was a text adventure game created in a group of 4. The game involved flying all over the planet, with a database of most of the world's airports allowing the player to select their next destination. It was developed in Python and the text was printed into the console. Random events could occur when the player arrived at a new airport. The game calculated money based on the distance flown, but deducted points for poor fuel economy.

Project 2

This project was an extension of the first one, developed by the same group. The game was made playable on the web using Python for the backend and JavaScript for the frontend. It featured a nice interface and a map API for the world map. The plane's path was calculated to reflect the curvature of the Earth. A login system saved players' progress in a database.

Current project

This project is a Space Invaders reboot developed in Unity. The game features both single-player and multiplayer modes, with a minimalistic look similar to the original Space Invaders but with modern touches. The multiplayer mode has players competing on a timer to get the highest score, sharing a grid of invaders that resets to one of five possible layouts when all aliens are defeated. The game is hosted as WebGL on itch.io. You can play it here.
